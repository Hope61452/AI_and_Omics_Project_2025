# AI and Omics Research Internship (2025)

This repository serves as the central storage for project assignments and computational workflows completed during the **AI & Omics Research Internship**. 

The primary focus of this work is the application of bioinformatics tools to solve complex biological problems, with a specific emphasis on high-throughput data analysis and transcriptomics.

## 🧬 Featured Project: Microarray Data Analysis
This project demonstrates a complete pipeline for analyzing gene expression data to identify significant biological markers.

### Key Computational Workflow:
* **Data Pre-processing:** Normalization and quality control of expression datasets using **R**.
* **Differential Expression Analysis:** Identifying statistically significant genes using the `limma` package.
* **Functional Interpretation:** Statistical analysis of gene expression patterns and fold-change visualizations.
* **Tools Used:** R, Bioconductor (limma, oligo), and ggplot2 for data visualization.
